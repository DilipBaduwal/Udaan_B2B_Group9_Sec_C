This database is modeled on Udaan, a B2B e‑commerce company. It contains eight interlinked datasets with sample records to support analysis and SQL operations.
The eight datasets are:
Customer data, Product, SKU,Logistics, Driver, Placed orders, Supplier data, Warehouse

These tables are connected through appropriate relationships, and sample data has been inserted into each one to enable queries, reporting, and further experimentation

File structure
Main folder:
database_model.mwb: MySQL Workbench model file containing the visual database design, including tables and relationships.​
schema.sql: SQL script used to create the full database schema (tables, keys, and other objects).​
IMage for scheme.

Data subfolder: nearly 16 folders with all the sql and xlsx file of inserted data.
*_insert.sql: SQL scripts with INSERT statements used to populate each table with data.​
*.xlsx: Excel files containing the same data as the insert scripts, provided for easy viewing and editing
